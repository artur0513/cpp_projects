Зажимая ЛКМ можно двигать камеру, ПКМ - добавить точку, колесико мыши - менять масштаб,
навести курсором на точку и нажать delete  - удалить точку
Можно использовать команды (курсор мигает внизу основного окна) - 
save filename - сохранить график в файл с заданным названием
load filename - загрузить график из файла с заданным названием
clear - удалить все точки
и еще, если график был загружен через команду load, то его можно сохранить в тот же файл с помощью ctrl + s
